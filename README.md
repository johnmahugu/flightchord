# Flight Chord

## Quick Setup

1. Clone the Repo
1. Update your ENV for URL's and DB access
1. Install Gems: `bundle install`
1. Run Migrations: `rails db:migrate`
1. Run Tests: `rails t`

Check the [Wiki](//github.com/shazm/flightchord/wiki) for more detailed information.
